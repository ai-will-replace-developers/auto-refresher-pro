## About This Project

A Chrome extension for multi-tab auto-refresh with random timing â€” randomly picks one tab and refreshes it at a time.

## Features

- **Multi-tab support** â€” Add multiple tabs to the refresh list
- **Random tab selection** â€” Each refresh cycle randomly picks one tab
- **One at a time** â€” Only refreshes one tab per cycle for natural behavior
- **Random interval refresh** â€” Set min/max delay, the extension picks a random time each cycle
- **Jitter** â€” Adds extra randomness to avoid predictable patterns
- **Cache bypass** â€” Option to force a fresh reload (no cache)
- **Quick presets** â€” Normal (300-700s), Relaxed (700-1000s), Slow (1000-1500s)
- **Live countdown** â€” See when the next refresh will happen
- **Next tab indicator** â€” Shows which tab will be refreshed next
- **Cycle counter** â€” Track how many refreshes have occurred
- **Auto cleanup** â€” Automatically removes tabs that are closed
- **Keyboard shortcuts** â€” Toggle with Alt+R, refresh now with Alt+Shift+R
- **Clean UI** â€” Fresh green/white theme with modern design

## Installation

1. Open `chrome://extensions` in Chrome
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the `Auto Refresher` folder
5. Pin the extension to your toolbar

## Usage

1. Navigate to a page you want to auto-refresh
2. Click the AutoRefresher icon
3. Click **Add Current Tab** to add it to the list
4. Repeat for other tabs you want to include
5. Set your desired min/max delay (or use a preset)
6. Click **Start**
7. The badge shows the number of tabs being managed
8. Each cycle, one random tab from your list will be refreshed
9. Click **Stop** to pause

## Shortcuts

- **Alt+R** â€” Toggle auto-refresh on/off
- **Alt+Shift+R** â€” Refresh a random tab immediately

## Notes

- Minimum delay is 3 seconds (Chrome may enforce higher minimums for alarms)
- Protected pages (chrome://, file://, etc.) cannot be added
- Closed tabs are automatically removed from the list
- If all tabs are closed, auto-refresh stops automatically
